Небольшой учебный проект.
Используется чистый HTML / CSS / JavaScript, без сторонних библиотек.

Проект реализует карточку с круговым индикатором прогресса, который можно:

-изменять числовым значением
-анимировать вращением
-скрывать и показывать содержимое карточки
-возвращать содержимое кнопкой Return
-Основная логика находится в классе ProgressBlock

ОСНОВНЫЕ ЭЛЕМЕНТЫ HTML

.card — контейнер карточки
.progress — круговой индикатор
.controls — панель управления
input[type="number"] — ввод значения прогресса
Animate — включает вращение
Hide — скрывает контент
Return — возвращает контент обратно


Основные методы JS:

normalize(value) — ограничивает значение от 0 до 100
drawRing(percent) — рисует круг прогресса
animateValueTo(target) — плавно изменяет значение
setHidden(isHidden) — скрывает/показывает контент
Инициализация происходит автоматически
